快速开始

此教程将让你快速安装并部署 Stuoe




安装Stuoe最简单的方法是使用配套的命令行

pip install stuoe

在使用stuoe startproject --name demobbs

stuoe startproject --name demobbs
                        cd demobbs

看下目录demobbs

使用Flask命令行启用项目(Flask已经在安装Stuoe预安装)

flask run --port 3000

打开浏览器，输入https://localhost:3000/install，配置论坛名称，Smtp服务器，初始化管理员信息
